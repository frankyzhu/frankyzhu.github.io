Java漫游笔记-01-环境配置

发表于 2015-06-23   |   分类于 java   |  

下载 JDK

前往 Oracle 的官方网站下载：
http://www.oracle.com/technetwork/java/javase/downloads/index.html

安装 JDK

建议：

  • 安装路径不要包含空格
  • 保留 JDK 版本号

例如：D:\dev\java\jdk1.8.0_25

设置执行路径

所谓执行路径，即操作系统搜索本地可执行文件的目录列表。

对于不同操作系统，设置方法有所不同。
对于 Windows 操作系统，操作路径为：系统属性 - 高级 - 环境变量 - 系统变量 。

  • 新建系统变量 JAVA_HOME
    • 变量值：D:\dev\java\jdk1.8.0_25（即 JDK 的安装目录）
  • 编辑系统变量 path
    • 在变量值头部插入：%JAVA_HOME%\bin;（不要漏掉末尾的分号“;”）
  • 新建系统变量 CLASSPATH
    • 变量值：.;%JAVA_HOME%\lib（不要漏掉前面的点号和分号“.;”）

测试

在 shell 或 cmd 窗口中执行：

如果配置正确，则会显示正确的 JDK 版本信息。

尝试使用 JDK

编写源代码

可以使用任意一种文本编辑器，编写源代码（ App.java 文件）

编译

使用 javac 命令将 App.java 文件，编译成字节码文件（ App.class 文件）

执行

使用 java 命令运行字节码文件

更多内容

  • 探索 JDK 安装目录下的内容
    • src.zip：Java 类库的源文件。
    • bin 目录：前面使用到的 javac 和 java 命令，实际上就是调用这里的 javac.exe 和 java.exe 工具。
    • lib 目录：JDK 工具用到的类库及其他文件。
